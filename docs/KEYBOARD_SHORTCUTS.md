# Keyboard Shortcuts

Linite features Vim-inspired keyboard navigation for power users.

## Getting Started

- Press `?` to open the keyboard shortcuts dialog

## Navigation

| Shortcut | Action |
|----------|--------|
| `/` | Focus search bar |
| `Esc` | Clear search / close dialog |
| `gg` | Scroll to top |
| `G` | Scroll to bottom |

## Selection

| Shortcut | Action |
|----------|--------|
| `Enter` / `Space` | Toggle selection |
| `Ctrl` + `Enter` | Select all filtered apps |
| `Ctrl` + `Space` | Deselect all apps |

## View Controls

| Shortcut | Action |
|----------|--------|
| `v` | Toggle view mode (grid/list) |
| `c` | Toggle category filter |
| `d` | Toggle distro selector |
| `s` | Toggle source preference |

## Actions

| Shortcut | Action |
|----------|--------|
| `g` | Open generate command dialog |
| `u` | Toggle install/uninstall mode |
| `i` | Open keyboard shortcuts dialog |
| `?` | Open keyboard shortcuts dialog |

## Mobile/Touch

- Keyboard shortcuts are primarily for desktop use
- Touch/tap interactions work on mobile devices
- Use the shortcuts dialog as reference
